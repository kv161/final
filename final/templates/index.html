Secure Text Transfer And Storge





How to use our web app


  • Register and generate your private key public key pair.

  • Download public key associated with the user with whom the file transfer is to take place.

  • Generate secret key from your private key and associated users public key

  • Encrypt the message using the generated secret key.

  • Upload the encrypted file.

  • User gets to choose from thelist of all the different files that are present on the cloud provided he has the required credentials for decrypting the file.

  • He can also download public key associated with the user with whom the file transfer is to take place.
